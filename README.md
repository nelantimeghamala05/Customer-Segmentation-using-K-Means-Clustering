# Customer Segmentation Using K-Means Clustering

## Project Overview

Customer Segmentation is a Data Science project that groups customers into different segments based on their purchasing behavior. This helps businesses understand their customers better and create targeted marketing strategies.

In this project, K-Means Clustering, an Unsupervised Machine Learning algorithm, is used to identify customer groups based on Annual Income and Spending Score.

---

## Problem Statement

Businesses often have customers with different spending habits and income levels. Treating all customers the same may reduce marketing effectiveness.

The objective of this project is to segment customers into meaningful groups so that businesses can:

* Understand customer behavior
* Improve customer engagement
* Create personalized marketing campaigns
* Increase sales and customer retention

---

## Dataset

Dataset: Mall Customer Segmentation Dataset

Features Used:

* CustomerID
* Gender
* Age
* Annual Income (k$)
* Spending Score (1-100)

Total Records: 200 Customers

## Machine Learning Technique

### K-Means Clustering

K-Means is an Unsupervised Machine Learning algorithm used to group similar data points into clusters.

The Elbow Method was used to determine the optimal number of clusters.

Optimal Number of Clusters (K): 5

---

## Results

The model successfully segmented customers into five distinct groups:

1. Premium Customers (High Income, High Spending)
2. Potential Customers (High Income, Low Spending)
3. Young Shoppers (Low Income, High Spending)
4. Budget Customers (Low Income, Low Spending)
5. Regular Customers (Average Income, Average Spending)

These customer segments can help businesses improve marketing strategies and customer targeting.
